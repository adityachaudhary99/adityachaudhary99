## ЁЯЪА Featured

### [0-suite](https://github.com/adityachaudhary99/0-suite) тАФ privacy-first browser tools ┬╖ [live demo](https://0-suite.vercel.app)
`dev0`, `web0`, `data0` & `privacy0` in one client-side Astro app: dev utilities, network lookups, DuckDB-WASM SQL over CSV/Parquet/JSON, and a PII scrubber. Nothing leaves the browser тАФ no uploads, no API keys, no tracking.
**Astro ┬╖ TypeScript ┬╖ WASM ┬╖ DuckDB**

### [contextdev](https://github.com/adityachaudhary99/contextdev-monitor) тАФ market landscape cartographer
Type a market category тЖТ an auto-built, **cited**, structured competitive landscape. Built on the context.dev API (discover тЖТ profile тЖТ synthesize); ships a pricing tracker, an honest extraction report-card, and a Claude-Code agent-skill.
**TypeScript ┬╖ Next.js**

### [yank](https://github.com/adityachaudhary99/yank) тАФ one command to download from anywhere
HTTP, cloud, git, media, torrents тАФ a single static Go binary with a native parallel engine (resume + checksums) that auto-dispatches to the right backend (rclone / git / yt-dlp / aria2c / curl).
**Go ┬╖ Cobra ┬╖ goreleaser**

### [smartloop](https://github.com/adityachaudhary99/smartloop) тАФ a loop controller for Claude Code
Reads each turn's transcript, judges *done / next / risk*, and drives autonomous iterations on a configurable autonomy dial (cautious / balanced / bold) behind a hard risk floor тАФ plus opt-in in-session auto-pilot hooks.
**TypeScript ┬╖ Claude Agent SDK**

### [MCP Atlas](https://github.com/adityachaudhary99/mcp-tool-deps) тАФ MCP tool-dependency graph
Discovers which MCP-server tools must run before others, within and across servers, and renders it as a scientific atlas. Heuristic + LLM extraction, entirely client-side.
**TypeScript ┬╖ SVG ┬╖ dagre**

### [Data File Viewer](https://github.com/adityachaudhary99/data-file-viewer) тАФ binary data explorer for VS Code / Cursor
Explore `pkl`, `h5`, `parquet`, `feather`, `arrow`, `avro`, `npy`, `mat` and more inline тАФ a tabular / array / hierarchical viewer + profiler backed by an isolated Python venv.
**TypeScript ┬╖ Python ┬╖ VS Code API**
